## Mô tả bài tập

Bài thực hành HTML cơ bản sử dụng các thẻ HTML5 Semantic để xây dựng:

* Trang hồ sơ cá nhân
* Trang giới thiệu sản phẩm
* Trang debug HTML
* Thực hiện validate mã nguồn bằng W3C Validator

---

## Các file chính

### profile.html

Trang hồ sơ cá nhân bao gồm:

* Header
* Navigation
* Ảnh đại diện
* Giới thiệu bản thân
* Bảng kỹ năng
* Thông tin liên hệ
* Footer

### products.html

Trang giới thiệu sản phẩm bao gồm:

* Danh sách sản phẩm
* Hình ảnh sản phẩm
* Giá bán
* Nút mua hàng
* Bảng so sánh cấu hình sản phẩm

### debug.html

Trang dùng để thực hành sửa lỗi HTML và kiểm tra tính hợp lệ của mã nguồn.
